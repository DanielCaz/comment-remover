# Comment Remover

Removes all comments from selected text.

## Features

It can remove different style of comments.

Java-like comments:

```java
// single line

/*
Multi line
*/
```

Python-like comments:

```python
# Single line
```

HTML-like comments:

```html
<!-- Multi line -->
```
